# Telco Customer Churn Analysis

This repository contains the Telco Customer Churn Analysis project.

## Project Overview
The project aims to:

- Identify why customers are leaving
- Predict which customers are likely to churn
- Perform univariate, bivariate, and trivariate analysis
- Create KPIs like Churn Rate and ARPU
- Develop customer segmentation (High-Risk and Loyal Customers)
- Provide business retention recommendations
- Build a basic churn prediction model in Python
- Build interactive dashboards in Power BI

## Repository Structure

- `Cleaned_Dataset.csv` — Preprocessed and ready-to-analyze dataset  
- `Python_Analysis.ipynb` — Data cleaning, EDA, KPI calculations, prediction model  
- `PowerBI_Dashboard.pbix` — Interactive dashboards for business insights  
- `Final_Report.pdf` — Full project report  

## Tools Used

- Python (Pandas, Seaborn, Scikit-learn)  
- Power BI  
- Excel (optional for preliminary analysis)
